Overview

AngularJS course covers the fundamental concepts of Angular like directives, routes, filters, services and factors which helps to build rich user interface web applications. This course also covers the highly efficient Angular features like data binding, scope management, form validation, routing, i18n & lot more. You will also get to implement an AngularJS mean stack-based project towards the end of the course, which gives you complete insights on the framework along with NodeJS.

Objectives

After the completion of the AngularJS course at Traingap, you will be able to:

  • Understand JavaScript MVC Framework
  • Understand Dependency Injection
  • Create Controllers,Understand Two Way Binding and Share data between Controllers
  • Learn about Directives, Filters, Services & Routes
  • Create custom Directive and Isolate scopes in Directive
  • Create Custom Filters
  • Understand why do we need modules
  • Understand usage of angular-ui, ng-grid and angular-translate

Suggested Attendees

  • This course is designed for professionals who aspire to gain expertise in building SPA and rich client applications by consuming REST API's.
  • Web Developers, Software Architects & Testing Professionals are the key beneficiaries of this course.

Prerequisites

  • The pre-requisites for this course includes basic programming experience and basic knowledge of JavaScript and HTML/HTML 5.

1. Introduction to JavaScript MVC Framework & AngularJS

  • In this module, you will understand the use of JavaScript MVC framework and why it is needed. You will also learn about the introduction to AngularJS SuperHeroic JavaScript Framework.
  • Topics - About AngularJS, Uses of AngularJS, Introduction to JavaScript MVC Framework, Write your first program in Angular JS, Comparison between JavaScript Frameworks.

2. Dependency Injection and Controllers

  • In this module, you will be understanding the dependency injection and ng-controller. You will also understand how to use nested controllers and share data between controllers.
  • Topics - Creating Controllers, Two Way Binding, Sharing data between Controllers, Inheritance, Dependency Injection.

3. Routing, Directive and Filters

  • In this module, you will understand different directives, filters and how to routing and create nested routing.
  • Topics - Why use Routing, Creating Routing, Nested Routes, Directives, Filters, Creating Directives and Filters.

4. Creating Custom Directives and Filters

  • In this module, you will understand the use of custom directive and filters in AngularJS application and you will learn to create custom directive and filters.
  • Topics - Custom Directives, Why Custom Directives, Creating and Using Custom Directives, Custom Filters, Creating and Using Custom Filters.

5. Third Party AngularJS Modules and Testing Angular

  • In this module, you will understand the useful AngularJS module available in the market and learn to use the modules in application.
  • Topics - Modules & its necessities, Usage of angular-ui, Usage of ng-grid, Usage of angular-translate and Testing AngularJS.

6. AngularJS with Node.js

  • In this module, you will understand the AngularJS directory structure, Introduction to Node.js, CURD operation with AngularJS and Node.js.
  • Topics - Introduction to Node.js, Introduction to Bower, Rest Exposure, Web Sockets and Push Notification.
