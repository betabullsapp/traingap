Overview

The iOS Development course will enable you learn how to build native applications for the iPhone and iPad using iOS 8 SDK, how to work with Xcode 6 and use Objective-C language to program your applications.

Objectives

After the completion of iOS App Development course at Traingap, you will be able to:

  • Build Cool iOS native applications
  • Understand Objective-C used for developing the applications
  • Use multiple Model-View-Controllers in an application
  • Use Storyboards for excellent UI design
  • Implement CoreData for data storage in an application
  • Implement Autolayout for different Screen sizes
  • Show Maps and access the location details in the application
  • Submit applications to AppStore

Suggested Audience

  • This course is designed for software programmers to understand iPhone and iPad application development on iOS using Objective-C.

Prerequisites

  • The only prerequisite for learning iOS Development is the knowledge on any object-oriented programming language.

1. Course Introduction, Overview of iOS, MVC Objective-C

  • Learning Objectives: - In this module, you will understand the basic overview of iOS, why and how you can develop applications for iOS. You will also learn the basic architecture and design patterns used in iOS.
  • Topics: - Description, History, Object Oriented Design Concept (MVC Architecture), Introduction to Apple Developer Site, Apple developer resources, Apple developer account.

2. Objective-C

  • Learning Objectives: - In this module, you will be able to learn the basic concepts of Objective-C language, how to and when to use categories and extensions while coding. You will also learn how to use the Xcode IDE efficiently.
  • Topics: - Introduction to Objective-C, Categories, Extensions, Working with Xcode IDE.

3. Foundation Framework, Attributed Strings

  • Learning Objectives: - In this module, you will learn the usage of Foundation framework for basic controls and also the use strings with a variety of requirements.
  • Topics: - Creating Objects, Dynamic Binding, Protocol, Collections and Data types, UIText View, View Controller Life Cycle, NSNotifications (Local Notifications).

4. Multiple MVC’s

  • Learning Objectives: - In this module, you will understand the principles of Object Oriented concepts used in View Controllers and learn to work with multiple MVC's in an application.
  • Topics: - Polymorphism in Controllers, UINavigationController, UITabBarController, Drawing Custom View, Handling Touch Gestures.

5. Blocks, Animations, Autolayout

  • Learning Objectives: - In this module, you will understand the use of blocks for delegation, call backs and animations. You will also learn to develop your screen for various device sizes.
  • Topics: - Passing a block, View Property Animation, Autorotation, Autolayout.

6. Multithreading, Scroll View

  • Learning Objectives: - In this module, you will implement multithreading along with scroll view for displaying large data on the screen.
  • Topics: - Posting blocks on queues, UIScrollView, UITableView, UISplitViewController, Device Specific UI Idioms.

7. Persistence, Documents & Core Data

  • Learning Objectives: - In this module, you will be able to persist data on the device using various methods. You will learn about Core Data framework and its usage along with the table view.
  • Topics: - Storing Model Permanently, File & Document Management, Core Data & Table View.

8. Core Location & Map Kit

  • Learning Objectives: - In this module, you will be able show maps on the screen along with the accessing user's location using Core Location framework.
  • Topics: - UIApplication, Network Activity Indicator, Maps (MapKit), Container View.

9. Modal Segue, Text Field, Alerts

  • Learning Objectives: - In this module, you will be able to show the alerts to the users, take input from the keyboard and hide the keyboard when the typing is done.
  • Topics: - Transitioning to a Controller, Text Input from User, Notifying the User, Camera, Taking Photo, Notifying User – Action Sheets, Application Lifecycle.

10. iCloud, Localization, Push Notifications

  • Learning Objectives: - In this module you will be able to synchornize your application to the iCloud, localize the application to various countries and languages and implement Push Notifications.
  • Topics: - Integration with iCloud, Internationalization, Push Notifications, Code-Signing of devices, Installing applications on the device, Submission of applications to the AppStore.
